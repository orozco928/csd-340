5 Things I’ve Learned Since the Course Started

  • HTML is the language that describes the structure of a webpage
  • A webpage will be made up of elements, which are enclosed using tags. ex <body>
  • CSS is the language that describes the way the content is shown
  • CSS 'rules' are made up of a selector(elements to be styled) and declaration (the particular stylings).
  • CSS declarations are made up of a property (which aspect of the element ex. color, font, border) and value(style setting (green, Arial, 10px).
